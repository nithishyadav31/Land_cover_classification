# 🧪 Land Cover Classification — Frontend

This is the React frontend for the Land Cover Classification project. It uses Vite for fast development and tailwind for styling.

## 🚀 Quick Start (Both Frontend & Backend)

For the full experience, you must run the backend and frontend together:

1. **Root Instructions**: See the main [README.md](../README.md) in the project root.
2. **Run Backend**:
   ```bash
   cd ../backend
   pip install -r requirements.txt
   python app.py
   ```
3. **Run Frontend**:
   ```bash
   npm run dev
   ```

## 🛠 Key Features
- **Interactive Global Map** (India Focused) using `react-leaflet`.
- **Direct MCNN Analysis** for real-time land cover classification.
- **Statistical Results Dashboard** with dominant class identification.
- **Integrated Pixel-wise Overlays** for visual verification.
